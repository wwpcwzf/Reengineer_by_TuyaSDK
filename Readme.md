# Reengineer_by_TuyaSDK
------------------------------------------------------------
目的：基于涂鸦蓝牙SDK灯DEMO开发LUTEC摄像头灯。
主要功能：
1、手机APP通过蓝牙给摄像头灯的摄像头模组配置WiFi。
2、支持蓝牙的标准协议、涂鸦协议和LUTEC自定义协议。
3、简单网关透传功能（LUTEC自定义协议）
    APP-->Internet-->XM-->Sigmesh
4、群组联动（LUTEC自定义协议）
5、PIR功能
    单/双探头，灵敏度可设。
6、CW调光
